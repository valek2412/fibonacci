# Fibonacci api

## Installation

```bash
1. $ git clone git@github.com:valek2412/fibonacci.git
2. $ cd fibonacci
3. $ yarn install
4. Create .env file like .env.example
```

## Usage

### Development
```bash
# development server with hot reload (nodemon)
$ yarn start:dev
```

### Production

```bash
# build for production
$ yarn build

# start production app
$ yarn start:prod
```

### Using docker

```bash
$ docker-compose up
```

### Run tests

```bash
$ yarn test
```
